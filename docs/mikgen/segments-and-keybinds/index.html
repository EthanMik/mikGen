Segments & Keybinds

A full reference for adding, selecting, and editing segments.

Adding Segments​

Similar segments share the same keybinds across every format, but each format only offers the segments its library supports; clicks for unsupported segments do nothing. See Format Differences for what each format includes.

ButtonSegmentDescription
Left ClickpointDriveDual PID algorithm, gets the robot to desired (x, y) as fast as possible
Ctrl + Left ClickposeDriveBoomerang algorithm, drives robot to desired (x, y, theta)
Alt + Left ClickdistanceDriveRelative motion, drives specified distance based on heading before it
Ctrl + Alt + Left ClickstrafeDriveSame as distanceDrive but moves left and right
Right ClickpointTurnPoints to drive segment in front of it
Ctrl + Right ClickangleTurnTurns to specified angle
Alt + Right ClickpointSwingSwings a drivetrain side to drive segment in front of it
Ctrl + Alt + Right ClickangleSwingTurns a side of drivetrain to specified angle

Two segments are added from buttons instead of clicks:

  • Start: added automatically the first time you click on an empty field. It sets the robot's initial pose.
  • Wait: pauses the path for a set time, added from the segment buttons in the path menu.

New segments are inserted after the currently selected segment, otherwise they are appended to the end of the path.

Selecting Segments​

ActionEffect
ClickSelect a segment
Ctrl + ClickAdd or remove a segment from the selection
Shift + ClickSelect a range in the path list
Click + Drag on fieldBox select
Ctrl + ASelect all
Ctrl + Shift + AInvert selection
EscDeselect all

Editing Segments​

ActionEffect
Drag waypointMove the segment on the field
Ctrl + DragSnap to grid (grid size set in Settings, default 0.5 in)
Arrow KeysNudge selected segments 0.25 in
Shift + Arrow KeysNudge selected segments 1 in
Shift + ScrollRotate heading in 90 degree snaps
Ctrl + Shift + ScrollRotate heading in 5 degree steps
Backspace / DeleteDelete selected segments
Drag row in path listReorder segments
Ctrl + ZUndo
Ctrl + YRedo

Waypoints also magnet-snap to nearby waypoints while dragging. The Mirror Horizontal and Mirror Vertical buttons in the bottom bar flip the selected segments across the field.

Each constant row in the path menu has options to reset it to default, set the current value as the new default, or apply it to every segment of the same kind.

Playback Controls​

ActionEffect
Space or KPlay / pause
J / LScrub backward / forward (hold Shift for 1 second steps)
RToggle robot visibility
Right Click a rowJump the simulation to that segment

The loop button next to the play controls replays the path continuously.
